# Changelog

All notable changes to the Blockchain Context Protocol specification will be documented in this file.

## [1.0.0] — 2026-07-11

### Added
- Initial public draft of the Blockchain Context Protocol (BCP) specification
- Four core layers: Context, Intent, Policy, Execution
- BCP Core Interface (minimal endpoints: context query, intent submission, policy validation, execution)
- BCP Extended Interface (multi-chain aggregation, simulation engines, route optimization, intent markets, proof framework)
- Security model covering authorization, policy enforcement, simulation, and verifiable proofs
- Interoperability notes for ERC-4337, ERC-6900, and ERC-8004 (proposed)
- Reference JSON examples for Context, Intent, Policy, and Execution Plan objects
- Published as a preprint on ResearchGate (DOI: 10.13140/RG.2.2.16419.00807)
